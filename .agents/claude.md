# Projekt: Orendt WM Tipprunde 2026

## Sprache
Antworte auf Deutsch.

## Was das ist
Internes WM-2026-Tippspiel für Orendt-Mitarbeiter. Self-Signup, Kicker-Punktesystem, Weltmeister-Sondertipp, Rangliste.

## Wichtige Befehle
- Dev: `npm run dev`
- Build: `npm run build`
- Test: `npm run lint`

## Konventionen
- Variablennamen Englisch, UI-Texte Deutsch
- Supabase-Tabellen mit Präfix `tip_`
- Design-Tokens aus DESIGN_SYSTEM.md / tailwind.config.js

## Memory
Lies zu Beginn jeder Session `.agents/memory/project.md` und `.agents/memory/current.md`.
Bei `update memory`: aktualisiere `current.md`, `project.md` nur bei Architektur-Änderungen.
